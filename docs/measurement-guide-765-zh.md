# 改装 Street Triple 765 完整测量指南

目标：把你这台**改装车**（换过 linkage / 狗骨 / YSS 避震 / fork position /
链调 / 轮胎 / 链轮）在 MotoSPEC Formula Explorer 里建成一套**全真实数据**
的 profile，达到与真实 MotoSPEC 选车后相同的计算能力（含换摇臂/狗骨）。

配套文档：测量点定义见 `measurement-points.md`（含七段距离协议的完整
细节）；等价类结论与 oracle 数据见 `research/triumph-765-motospec.md`。

**3D 扫描替代路线**：取点图见 [`scan-points-zh.svg`](scan-points-zh.svg)
（车头朝左的侧视投影，含连杆放大图）。如果能借到 3D 扫描仪，**一次
扫描（悬挂完全伸展、含地面）替代本指南 Session A + B + C 的全部几何
测量**——rake / WB / Rf / β / offset / 全部连杆坐标都从标记点坐标直接
导出，免七段距离法和倾角仪。Session D（称重）无论哪条路线都要单独做。

## 0. 五张截图已经给了什么、还缺什么

| 截图内容 | 对你的改装车还有用吗 |
|---|---|
| 主表三列（offset 26/27.5/29 的 RESULTS） | ✅ 已用于验证我们的公式=MotoSPEC 公式（trail/轴距差量精确复现）。**不再需要测的东西** |
| LINK DIMENSIONS（原厂连杆杆长） | ⚠️ 只描述**原厂**连杆——你换了摇臂和狗骨，这些数字对你的车作废；仅当你装回原厂件时有用 |
| MR 曲线图 | ✅ 钉住了原厂连杆的拟合；你的连杆需要自己的数据（Session A） |
| 赛事支援规格表（叉长 730、行程、弹簧、sag 目标） | ✅ 硬件固有值仍有效（原厂件的）；YSS 的规格要另测/查 |
| Wheel rate | ⚠️ 前轮 rate 32.37 的公式尚未破解（已知模型差异），只作参考，**测什么都补不上** |
| 车架基线（rake 23.7 / WB 1414.3 / β 12.23 / Rf 304.6） | ❌ **对你的车全部失效**——轮胎、前叉位置、后端全变了，姿态类数字必须重测 |

一句话：截图的遗产是"公式已验证 + 原厂件规格"；你的车的**几何基线和
连杆坐标全部要自己测**。好消息：一个下午 + 常规工具就能做完。

## 1. 参考状态（Reference Setup）——所有姿态测量的统一要求

Session B 和 C 的每一个数字都必须在**同一个冻结状态**下测：

- **水平地面**（水平仪验证），整车**竖直**（需要助手扶正，前轮朝正前）
- **前后悬挂完全伸展（0 压缩）**：
  - 前：提起车头直到前叉完全顶出，前轮仍触地
  - 后：提起车尾直到避震完全顶出，后轮仍触地
  - 预载不影响完全伸展的位置（机械限位决定），不用调
- **胎压打到你日常使用值**（影响轮径 → 影响一切姿态）
- 当天车上装着什么就是什么：fork position、offset、YSS、狗骨、链调
  ——**全部如实记录，它们就是你的 `*_ref` 字段**
- **整个测量日不得改动任何设定**（尤其链调和后端任何零件——A/B 耦合条款）

Session A 不依赖姿态（纯点到点距离），但要求后轮离地、避震顶出（机构
成为刚体）且不碰链调。

## 2. 工具清单

| 工具 | 用途 |
|---|---|
| 数显卡尺（150 mm；最好另有 300 mm 长爪或卡规） | 杆长、短距离，孔心到孔心 |
| 钢直尺 / 卡规接杆（400 mm 级） | L6/L7（约 300–400 mm，卷尺有挠度） |
| 卷尺（3 m） | 轴距 |
| 数显倾角仪（角度盒） | rake（贴前叉管） |
| 水平仪 | 地面与车身垂直验证 |
| 卷尺/高度尺 | 轴心离地高（h①、h②、前轴高） |
| 摇臂起落架（后驻车架） | Session A |
| 2–3 人 | 扶车、提车、读数 |

精度目标：距离 ±0.5 mm，角度 ±0.1°。所有孔位量**螺栓/轴承中心**。

## 4. Session B — 一个角度（β_ref），5 分钟

参考状态下（§1，后悬挂完全伸展、后轮触地）：

| 量什么 | 记录 (mm) |
|---|---|
| h① = 摇臂枢轴中心离地高 | ______ |
| h② = 后轴中心离地高 | ______ |

`β_ref = asin((h① − h②) / L1)` → app 的 `beta_static`。

**耦合条款**：B 与 A 之间后端零件与链调不得有任何改动（⑥⑦ 坐标的
旋转角就是这个 β）。

## 5. Session C — 整车基线（与 B 同一天同一状态做完）

| 量什么 | 方法 | → app 字段 |
|---|---|---|
| Rake | 倾角仪贴前叉管外壁（BPF 叉管平行于转向轴），车竖直、轮正 | `Rake_Static` |
| 轴距 | 前后轴心水平距（卷尺，两侧各量一次取平均） | `WB` |
| 前轴心离地高 | 高度尺/卷尺 | `Rf` |
| Yoke offset | 零件规格/刻印，或量转向轴到前叉管轴线垂距 | `Yoke_Offset`(=ref) |
| Fork position | 叉管顶出上三星台的长度（卡尺深度尺） | `Fork_Position`(=ref) |
| 链轮齿数 | 数 | `Front/Rear_Sprocket` |
| 链条节距 | 链条型号（520=15.875） | `Chain_Pitch` |

自动生成/换算的 ref：`Swingarm_Length_ref` = L1、`Shock_Length_ref` =
YSS eye-to-eye、`Yoke_Offset_ref`、`Fork_Position_ref` 同上。

## 6. Session D（可选）— 解锁 Anti-Squat 与配重

| 量什么 | 方法 | → 字段 |
|---|---|---|
| 前链轮位置 | 前链轮（countershaft）中心 ↔ 摇臂枢轴中心的水平/垂直距（车架属性，量一次永久） | `Front_Sprocket_X/Y` |
| 前后轮荷 | 两个体重秤/角秤，水平地面，含骑手赛车姿势 | `Mass`、weight dist、`L_CG` |
| CG 高 | 垫高后轮 ≥150 mm 重称前轮（见 measurement-points.md 公式） | `H_CG` |

不做 D：Anti-Squat 与配重显示诚实空白，其余全部可用。

## 7. 数据回来之后的流程

1. 把 L1–L8 + 消歧侧别发给助手/脚本 → 反算 ③⑥⑦ 坐标（两圆相交，
   app 的 lengths-only 模式同款数学），用 β_ref 转到车架坐标轴。
2. Linkage Setup 页：lengths-only 模式输入 ③⑥⑦ 锚点 + §3.2 四段
   杆长 → 存为你的 linkage profile（如 `my-765-race-link`）。
3. Chassis Setup 页：填 §4/§5 全部字段 → 存为你的 chassis profile
   （如 `my-765-race`）。ref 与 live 同值（基线日状态）。
4. 组件目录：YSS 避震建条目（长度/行程/弹簧），前叉沿用 showa-bpf-765
   或按你的弹簧建变体。
5. Data Table 选你的 chassis + linkage + 组件 → RESULTS 全绿全真实。
   之后调 offset / fork position / 链调 / 换避震长度直接在表里改；
   **换摇臂/狗骨也能算了**——改 linkage profile 里的杆长即可（③⑥⑦
   锚点不变）。

## 8. 常见错误

- 量孔缘不量孔心（差一个半径，毫米级全毁）
- Session A 期间碰了链调（② 移位，L3/L5/L7 互相矛盾）
- A 和 B 之间换了后端零件（β 旋转角作废 → ⑥⑦ 坐标系统性偏转）
- 姿态测量时悬挂没有完全顶出（sag 零点错 → 所有 ref 错）
- 车不竖直/地不水平（rake 和离地高全偏）
- 轮胎气压随手测（轮径变 → Rf、β、rake 全联动）
- 把 WB 当可调参数手改（WB 是 ref 状态的一次性测量，链调走
  `Swingarm_Length`，offset 走 `Yoke_Offset`，永远不手改 WB）
